# Pick a Sneak — Frontend

A sneaker e-commerce catalog frontend, built as a ground-up redevelopment of my Zuitt Bootcamp Capstone 3 project. The original build no longer exists — this is a full rebuild on a more structured, industry-aligned stack and approach.

Backend: [`api-pick-a-sneak`](https://github.com/dev-yangg/api-pick-a-sneak) — a separate Express/MongoDB REST API. This frontend currently runs on static/mock data while API integration is in progress.

## Status

🚧 **Work in progress**

## Stack

- React
- TypeScript
- React Router v7 (Data mode)
- Tailwind CSS
- Vite

## Why this rebuild

The original Zuitt Bootcamp Capstone 3 project no longer exists. It was built with Create React App, CSS styling, and JavaScript. This version is built with Vite, React, Tailwind CSS, and TypeScript.
